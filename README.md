# homebrew-tap

## Installation

Install the tap:

```sh
brew tap mininit/tap
```

Install an individual cask or formula using:

```sh
brew install mininit/tap/<name>
```

For more information on installing Homebrew https://brew.sh and `brew tap` https://docs.brew.sh/Taps

## Available packages

# Formulae
